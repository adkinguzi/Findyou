# Findyou
一款识别页面源码敏感字的谷歌插件
## 编写插件的想法💡
一方面单纯的冲动，一直有这么一个想法。另一方面是平时测试部分网站被挂暗链，需要查看页面源码不是很方便，所以有了插件。
项目断断续续写了两三天，代码属于东拼西凑，像个渣男。感谢大晚上给我指导的朋友👬。
## 安装
谷歌浏览器 输入地址 chrome://extensions/
打开开发者模式，然后加载下载的项目即可。
